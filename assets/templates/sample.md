COLOR PALETTE
--------------------------------------------------

Design Goal:
Create a calm, trustworthy, and accessible visual identity suitable for a
healthcare-focused platform, with special attention to comfort and clarity.

1. Primary Color
Name: Soft Rose
Hex: #E85C7A
Usage:
- Primary buttons
- Active states
- Key highlights
- Links and CTAs

2. Secondary Color
Name: Lavender Gray
Hex: #6B6F9E
Usage:
- Secondary buttons
- Section headers
- Icons

3. Accent Color
Name: Mint Calm
Hex: #5FB3A2
Usage:
- Success states
- Subtle highlights
- Badges and indicators

4. Background Colors
Light Background: #FFF7F9
Dark Background: #1C1C24
Surface / Card Color (Light): #FFFFFF
Surface / Card Color (Dark): #2A2A35

5. Text Colors
Primary Text (Light): #1F1F1F
Secondary Text (Light): #555555
Primary Text (Dark): #F5F5F5
Secondary Text (Dark): #C7C7C7
Muted / Disabled Text: #9A9A9A

Accessibility Notes:
- All text/background combinations maintain WCAG AA contrast standards
- Accent colors are never used alone for critical information

--------------------------------------------------
TYPOGRAPHY SYSTEM
--------------------------------------------------

Design Goal:
Ensure readability, emotional softness, and consistency across all devices.

Primary Font (UI & Body):
Font Name: Inter
Weights Used: 400 (Regular), 500 (Medium), 600 (SemiBold)
Reason:
- Excellent legibility
- Designed for screens
- Neutral and professional tone

Secondary Font (Headings):
Font Name: Poppins
Weights Used: 500, 600
Reason:
- Friendly and modern
- Adds personality without harming readability

Font Hierarchy:
H1: 32px / SemiBold / Poppins
H2: 28px / SemiBold / Poppins
H3: 24px / Medium / Poppins
H4: 20px / Medium / Inter
H5: 18px / Medium / Inter
H6: 16px / Medium / Inter

Body Text:
Size: 14–16px
Weight: Regular (400)
Font: Inter

Caption / Helper Text:
Size: 12px
Weight: Regular
Color: Secondary / Muted Text

Responsive Notes:
- Font sizes scale down smoothly on mobile
- Line height maintained at ~1.5 for readability

--------------------------------------------------
DESIGN RATIONALE
--------------------------------------------------

The color palette prioritizes calmness and trust, avoiding harsh contrasts
while maintaining accessibility. Typography choices balance professionalism
with approachability, ensuring FEM-CARE feels supportive rather than clinical.

This system is intentionally minimal to allow future scalability without
design inconsistency.
